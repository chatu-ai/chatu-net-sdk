# ChatU-net-sdk

ChatUAISDK.Console 是命令行示例

## 使用方法

修改 ChatUAISDK.Console  的 Program.cs 文件中的配置

```csharp

var accessToken = "你的AccessToken";

```

之后运行即可。

```
AccessToken 请于 https://agent.chatu.ai 获取
```
